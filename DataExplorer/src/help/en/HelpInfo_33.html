"Graphics Templates" Menu

Using this menu graphics templates are managed. Graphics templates contains graphics visualization properties.

If available and applicable a graphics template will applied according the device and channel configuration. Graphics templates contains properties regarding the once adjusted by the contect menu of the curve selector, like line color, number format, ... The directory to the graphic template files "GraphicTemplates" under the application configuration path.

  • "Save Graphics Template" stores a template according channel configuration and device (DeviceName_ChannelConfigurationNumber.xml)
  • "Save Graphics Template as ..." opens a file save dialog to save a graphics template with a given name
  • "Load Graphics Template" opens a open file dialog to select a graphics template to be applied
