Statistics View Window

The statistics window shows statistic values for all measurements which can be displayed graphically. The device properties file hold all configuration required to calculate statistic values. Configure a trigger will enable to reference all statistic calculation to the measurement point within. In addition a trigger will enable to summarize all maxima within the trigger area. This is shown in the sample screen shot below by height. Declaring a comment for each line of measurement it is possible to make clear referencing trigger.

Probably it is not required to have dialog to configure the statistic calculation for each device. This is the reason to have this dialog enabled for selected devices and measurements where it seams required.
