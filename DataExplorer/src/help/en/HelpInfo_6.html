Table View Window

The table is showing all data even if a curve is invisible. Depending on the amount of data the size of the table can be real big.

The calculation and formatting consumes time and memory. Big amount of data making the table unclear. This is the reason to have the capability to disable this calculation individual for each device (visibility configuration).
