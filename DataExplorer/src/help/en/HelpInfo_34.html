"View" Menu

With this menu the curve selector can be hide or made visible. In addition a graphics header and dataset comment can hide or made visible.



"Help" Menu

Using the help menu this text can be reached and the about diaolg can be opened to check the version number of this application. Selecting the version check the home page of the OpenSerialDataExplorer can be reached to check if a newer version is available.
