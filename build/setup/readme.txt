There are some preferences files to get global settings, compiler settings and code format in sync with all eclipse development environments.
the three files has to be imported :
 1) global stuff eclipse_preferences.xml (Eclipse menu File-> Import-> Generic-> Preferences-> select import all
 2) code_cleanup_preferences.xml (Eclipse menu Window-> Preferences-> Java-> Code Style-> Cleanup-> Import
 3) code_format_preferences.xml (Eclipse menu Window-> Preferences-> Java-> Code Style-> Formatter-> Import
