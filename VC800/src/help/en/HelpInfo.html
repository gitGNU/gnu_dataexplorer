VC8x0 Device Dialog Introduction



The VC8x0 device dialog meight be used to start and stop data collection. It is showing information about the adjusted measurent type and batterie condition.

Usage

Afterwards the serial communication port adjustments are done using the device selection dialog the start button can be used to initiate data collection. Nevertheless the "start gathering" button can be used instead without openeing this dialog.

Switching the measurement type at the multimeter itself will create a new data set.

Pushing the "stop" button will stop gathering data.
